# DATA221_Assignment_2
Assignment 2 for DATA221 - Introduction to Data Science

# File Information

1. Data221_Assignment2_Question1.py: 
Reads 'sample-file.txt', cleans the text as per the requirements, and prints the 10 most frequent words.

2. Data221_Assignment2_Question2.py: 
Constructs bigrams from the text in 'sample-file.txt' and prints the 5 most frequent bigrams.

3. Data221_Assignment2_Question3.py: 
Detects near-duplicate lines in a text file after removing punctuation, whitespace, and case differences.

4. Data221_Assignment2_Question4.py: 
Filters student data based on study time, internet access, and absences, saves the filtered records, and prints summary statistics.

5. Data221_Assignment2_Question5.py: 
Creates grade bands based on the grades column and generates a grouped summary table with student counts, average absences, and internet access percentages.

6. Data221_Assignment2_Question6.py: 
Categorizes crime data into high and low crime risk groups and compares average unemployment rates between them.

7. Data221_Assignment2_Question7.py: 
Get the data from Wikipedia page (topic Data Science) to extract and print the page title and the first paragraph with more than 25 words.

8. Data221_Assignment2_Question8.py: 
Extracts valid section headings from the Wikipedia page on Data Science and saves it to a text file.

9. Data221_Assignment2_Question9.py: 
Get the data from Wikipedia page (topic Machine Learning) and, extracts the first valid table that has least three data rows, and saves it as a CSV file.

10. Data221_Assignment2_Question10.py: 
Implements a function that take in file name and keyword to find lines containing a keyword in a text file and prints matching results.

# Input Data Files

1. sample-file.txt: 
Text file used for Questions 1, 2, 3, and 10 for word counting, bigram, duplicate detection, and keyword searching.

2. student.csv: 
Student dataset used for Questions 4 and 5 to analyze student grades, absences, and internet access.

3. crime.csv: 
Crime dataset used in Question 6 to compare unemployment rates across different crime risk levels.

# Output Files

1. high_engagement.csv: 
Output of Question 4, containing students with high study time, internet access, and low absences.

2. student_bands.csv: 
Output of Question 5, containing a summary table that includes student counts, average absences, and internet access percentages grouped by grade band.

3. headings.txt: 
Output of Question 8, listing cleaned section headings extracted from the Wikipedia page on Data Science.

4. wiki_table.csv: 
Output of Question 9, containing table extracted from the Wikipedia page on Machine Learning.
